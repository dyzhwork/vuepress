## 1. 安装

Windows 系统安装 NodeJS 简单的不要,只需要双击运行程序,下一步->下一步->....最后就完成安装。

## 2. 检验安装 node 是否成功

```bash
node -v
```

返回相应的版本信息即为安装成功,如:v14.21.3

## 3. 检验 npm 是否安装成功

```bash
npm -v
```

返回相应的版本信息即为安装成功,如:6.14.18
