## react 学习

### 1. 组件

#### 1.1 组件的分类

- 函数式组件
  - 函数式组件，不需要使用 class 关键字，直接一个函数，函数名就是组件的名字。
  - 函数式组件的写法，可以简化代码，但是，函数式组件没有生命周期，也没有 this 对象，不能访问 this.setState()，不能访问 this.props 属性，也不能访问组件的生命周期方法。
